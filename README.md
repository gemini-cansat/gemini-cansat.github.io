# Blog Website

This project is a simple blog page that includes a contact section for users to ask questions. 

## Project Structure

- `src/index.html`: The main HTML page of the blog.
- `src/css/styles.css`: Styles for the blog page.
- `src/js/contact.js`: JavaScript for handling the contact form submission.
- `.gitignore`: Specifies files and directories to be ignored by Git.

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd blog-website
   ```

3. Open `src/index.html` in your web browser to view the blog page.

## Contact Form

The contact form at the bottom of the page allows users to send questions via email. Ensure that the email functionality is properly configured in `src/js/contact.js`.
